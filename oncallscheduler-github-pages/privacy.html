プライバシーポリシー

OnCall Schedulerにおける個人情報の取り扱いについて

トップ サポート

1. はじめに

OnCall Scheduler（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。 本プライバシーポリシーは、本アプリにおける情報の取得、利用、管理について説明するものです。

2. 取得する情報

本アプリでは、以下の情報を取得する場合があります。

  • メールアドレス
  • ユーザー名または表示名
  • ログイン認証に必要な情報
  • グループ情報
  • 登録された希望日、当直、オンコール、救急ローテーション等のスケジュール情報
  • Googleカレンダー連携に必要な認証情報およびカレンダー情報
  • 端末情報、アプリ利用状況、不具合解析に必要な情報

3. 利用目的

取得した情報は、以下の目的で利用します。

  • 本アプリの機能提供のため
  • ユーザー認証およびアカウント管理のため
  • グループ内でのスケジュール共有・調整のため
  • Googleカレンダーとの連携機能を提供するため
  • 不具合対応、問い合わせ対応、サービス改善のため

4. 第三者サービスの利用

本アプリでは、認証、データ保存、解析、カレンダー連携等の目的で、以下の第三者サービスを利用する場合があります。

  • Firebase Authentication
  • Firebase Firestore
  • Firebase Crashlytics
  • Google Sign-In
  • Google Calendar API

これらのサービスにおける情報の取り扱いについては、各サービス提供者のプライバシーポリシーをご確認ください。

5. 第三者提供

本アプリは、法令に基づく場合を除き、ユーザーの同意なく個人情報を第三者に提供することはありません。

6. 情報の管理

取得した情報について、漏えい、滅失、改ざん等を防止するため、適切な安全管理措置を講じます。

7. アカウント削除・データ削除

ユーザーは、アプリ内の機能またはお問い合わせにより、アカウント削除や関連データの削除を依頼できます。 削除を希望される場合は、サポート窓口までご連絡ください。

8. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて変更されることがあります。 変更後の内容は、本ページに掲載された時点で有効となります。
