# OnCall Scheduler GitHub Pages

This repository contains a simple GitHub Pages website for OnCall Scheduler.

## Pages

- `index.html`: App introduction page
- `support.html`: Support URL page for App Store Connect
- `privacy.html`: Privacy Policy URL page for App Store Connect
- `style.css`: Shared stylesheet

## App Store Connect URLs

After publishing with GitHub Pages, use these URLs:

- Support URL: `https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/support.html`
- Privacy Policy URL: `https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/privacy.html`

If you use a custom domain, examples:

- Support URL: `https://oncallscheduler.app/support.html`
- Privacy Policy URL: `https://oncallscheduler.app/privacy.html`
